Docker Options Information

Docker options allow you to customize how containers are built, deployed, and run for your application.

  • Build Options: Applied during docker build. Control how your Docker image is constructed.
  • Deploy Options: Applied when the container is deployed to your environment.
  • Run Options: Applied when your container is started with docker run.

Common examples:

--memory 512m --ulimit nofile=4096 --shm-size 256m --cpus 0.5
